# 🚦 智慧交通智能体系统（Smart Traffic Agent System）

> 智慧交通 × 人工智能 × 多模态感知
> 以大模型为核心驱动，构建面向城市交通治理、道路安全与出行服务的智能体平台。

---

## 🧭 项目简介

**智慧交通智能体系统** 是一个基于大模型的综合性交通智能体项目，
旨在通过自然语言理解、实时数据分析与多源感知融合，实现对交通运行状态的智能识别、预测与决策支持。

系统结合了 **AI Agent 框架、知识图谱、物联网数据流** 与 **可视化管理平台**，
为智慧城市提供一体化的交通感知与治理解决方案。

---

## ✨ 核心功能

| 模块              | 功能说明                                                   |
| --------------- | ------------------------------------------------------ |
| 🚗 **交通态势分析**   | 基于摄像头 / GPS / 信号灯数据，自动识别道路拥堵、事故、流量波动等。                 |
| 🧠 **智能决策引擎**   | 使用大模型（LLM）与规则融合，实现事件推理与调度建议生成。                         |
| 📊 **交通数据可视化**  | 提供实时交通流量、车速热力图、道路运行趋势图等可视化看板。                          |
| 🗺️ **多源数据接入**  | 支持 IoT 设备、视频流、卫星定位、气象数据等多源输入。                          |
| 💬 **自然语言交互**   | 用户可通过自然语言查询交通状况、生成分析报告或调度建议。                           |
| 🧩 **插件化智能体架构** | 支持自定义 Agent，例如“信号灯优化 Agent”、“拥堵预测 Agent”、“出行规划 Agent”。 |

## 🚀 快速启动

```bash
# 克隆项目
git clone git@github.com:cxjchelsea/aiDoctor.git
cd aiDoctor

# 安装依赖
pip install -r requirements.txt

# 启动后端服务
python run_server.py

# 启动前端（如有）
```

* API 服务：`http://localhost:8000`
* Web 前端：`http://localhost:3000`

## 🧠 模型与智能体说明

* **主模型**：基于 BERT / Qwen 的意图识别与多智能体任务调度模型；
* **训练数据**：交通场景问答语料 + 交通事件分类样本；
* **微调方式**：LoRA + 多任务联合训练；
* **目标智能体示例**：

  * `TrafficMonitorAgent`：实时流量分析；
  * `AccidentPredictAgent`：事故风险预测；
  * `SignalOptimizeAgent`：信号灯配时优化；
  * `RoutePlanAgent`：个性化出行推荐。
